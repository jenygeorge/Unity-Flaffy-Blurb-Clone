# Unity Flappy Bird Clone

A Unity-based Flaffy Blurb clone with drifting cloud parallax effects and prefab-based spawning.

## How to Run
1. Clone this repo
2. Open in Unity (tested with Unity 2022.x)
3. Run the `Scenes/MainScene.unity`

## Features
- Cloud spawner with randomized speed and scale
- Parallax background for visual polish
- Prefab-based pipes and obstacles
